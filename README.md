# Final Project

## Description
This project analyzes state-specific influenza vaccination coverage among women with live births using PRAMS data.

## How to run
To generate the final report, run:

make report

## Structure
- data/: contains the dataset used in the analysis
- code/: contains the R script used for analysis
- output/: contains tables and figures
- report/: contains the final R Markdown report
